# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Debuggix, please report it responsibly. **Do NOT open a public GitHub issue.**

Instead, email us at: **security@debuggix.com**

We will respond within 48 hours with:
- Confirmation of receipt.
- An estimated timeline for a fix.
- Credit for the discovery.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Beta (current) | ✅ Full support |

## Security Best Practices
1. Review AI-generated fixes before implementing.
2. Never commit API keys or secrets found by the scanner.
